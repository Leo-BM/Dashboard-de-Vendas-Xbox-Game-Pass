# Desafio: Dashboard de Vendas Xbox Game Pass

Este projeto transforma dados brutos de assinaturas em um dashboard executivo no Excel para análise de desempenho.

## Estrutura do Excel
- **Dashboard**: Painel visual com KPIs (Faturamento, Assinantes, Ticket Médio) e gráficos de tendência e participação.
- **Data**: Base de dados tratada e limpa.
- **Calculations**: Tabelas auxiliares para os gráficos.

## Insights Analisados
1. O faturamento é dominado pelo plano **Ultimate**.
2. Existe uma correlação entre o uso de cupons e o volume de novas assinaturas.
3. Os Add-ons (EA Play e Minecraft) representam uma fatia importante da receita incremental.
